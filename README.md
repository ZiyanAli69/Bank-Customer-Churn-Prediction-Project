\# Bank Customer Churn Prediction



\##  Overview

This project focuses on predicting customer churn in a banking dataset using machine learning techniques. The goal is to identify customers who are likely to leave the bank based on behavioral and demographic features.



\##  Dataset

\- 10,000 customer records

\- Includes demographic details, account balance, transaction behavior, and credit information



\##  Tools \& Libraries

\- pandas

\- NumPy

\- matplotlib

\- seaborn

\- scikit-learn

\- StandardScaler

\- LabelEncoder

\- GridSearchCV

\- XGBoost



\##  Project Workflow

1\. Data Cleaning \& Preprocessing

&nbsp;  - Handled missing values

&nbsp;  - Encoded categorical variables using LabelEncoder

&nbsp;  - Applied StandardScaler for feature scaling



2\. Exploratory Data Analysis

&nbsp;  - Visualized churn distribution

&nbsp;  - Analyzed correlation between features

&nbsp;  - Identified churn patterns using seaborn \& matplotlib



3\. Model Building

&nbsp;  - Logistic Regression

&nbsp;  - Random Forest

&nbsp;  - XGBoost



4\. Model Evaluation

&nbsp;  - Confusion Matrix

&nbsp;  - Accuracy Score

&nbsp;  - Cross-validation



5\. Hyperparameter Optimization

&nbsp;  - Used RandomizedSearchCV

&nbsp;  - Optimized XGBoost parameters

&nbsp;  - Achieved \*\*86.5% test accuracy\*\*



\##  Results

\- Best Model: XGBoost

\- Accuracy: 86.5%

\- Improved prediction stability through hyperparameter tuning
